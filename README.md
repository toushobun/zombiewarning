# Zombie Warning

## 概要
学生時代に作成した Python 製の簡単なゲームです。
プレイヤーを操作し、ゾンビを避けながらできるだけ長く生き残り、スコアを伸ばすことを目的とした小規模な作品です。

Python の基礎的なプログラミング、ゲームループの実装、画像・音声ファイルの扱い、スコア管理などの学習を目的として作成しました。

## 使用技術
- Python
- Pygame
- 画像 / 音声アセットのローカル読み込み

## 主な機能
- プレイヤー操作
- ゾンビの出現および判定処理
- スコア記録
- ゲームオーバー処理
- 画像・音声リソースの読み込み

## ファイル構成
- FirstGame01.py : メイン処理
- Monster/ : 敵関連の素材
- Player/ : プレイヤー関連の素材
- buttons/ : ボタン素材
- img/ : 画像素材
- music/ : 音声素材
- highestScore.txt : ハイスコア保存用ファイル
- requirements.txt : 必要ライブラリ一覧

## 実行方法
1. Python をインストール
2. 必要ライブラリをインストール

pip install -r requirements.txt

3. 実行

python FirstGame01.py

- 学生時代に作成した小規模な個人制作です。
